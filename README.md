# USA Properties - Casas en Renta

Página web para mostrar propiedades disponibles para renta en Estados Unidos.

## Características

- Diseño moderno y responsivo
- Galería de propiedades con imágenes
- Formulario de contacto
- Información de servicios
- Compatible con dispositivos móviles

## Archivos

- `index.html` - Página principal
- `styles.css` - Estilos CSS
- `script.js` - Funcionalidades JavaScript

## Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador

## Personalización

Para agregar tus propiedades:
1. Edita `index.html` y modifica las tarjetas de propiedades
2. Reemplaza las imágenes con URLs de tus fotos
3. Actualiza los datos de contacto
